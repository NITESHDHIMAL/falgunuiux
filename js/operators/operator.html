Operator types

  1. Arithmetic Operator
    1. + : Addition operator.
    2. - : Subtraction operator.
    3. * : Multiplication operator.
    4. / : Division Operator
    5. % : Modulus / Remainder Operator
    6. ++ : increment
    7. -- : decrement
    8. **
  2. Asignment Operator
    1. =
    2. -=
    3. +=
    4. -=
    5. *=
    6. /=
    7. %=
    8. **=
  3. Comparision Operator
    1. ==(equal value)
    2. === (Strong comparison/data type + value)
    3. !=(Not equal to)
    4. !==(not equal value + data type)
    5. < (less than)
    6. > (greater than)
    7. >= (greater than or equal to)
    8. <= (greater than or equal to)
  4. Logical Operator
    1. &&
    2. ||
